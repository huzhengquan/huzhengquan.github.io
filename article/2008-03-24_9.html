用Acid3测测你的浏览器

Acid3是最新的浏览器规范测试工具，测试项有：

DOM2 Core DOM2 Events DOM2 HTML DOM2 Range DOM2 Style (getComputedStyle, …) DOM2 Traversal (NodeIterator, TreeWalker) DOM2 Views (defaultView) ECMAScript HTML4 (<object>, <iframe>, …) HTTP (Content-Type, 404, …) Media Queries Selectors (:lang, :nth-child(), combinators, dynamic changes, …) XHTML 1.0 CSS2 (@font-face) CSS2.1 (inline-block, pre-wrap, parsing…) CSS3 Color (rgba(), hsla(), …) CSS3 UI (cursor) data: URIs SVG (SVG Animation, SVG Fonts, …)
